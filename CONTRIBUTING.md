# Contributing to PromptForge AI

Thank you for your interest in contributing! 🎉

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/<your-username>/promptforge-ai.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make changes** and write clear commit messages
5. **Push** and open a **Pull Request**

## Code Style

- Use `camelCase` for variables and functions
- Use meaningful names — no single-letter variables
- Add comments for complex logic
- Keep functions small and focused

## Commit Convention

Follow [Conventional Commits](https://conventionalcommits.org):

```
feat: add new feature
fix: fix a bug
docs: update documentation
style: formatting changes
refactor: code refactoring
test: add tests
chore: maintenance
```

## Reporting Bugs

Open an [issue](https://github.com/iamjagadishkumar24/promptforge-ai/issues) with:
- A clear title
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Code of Conduct

Be respectful, inclusive, and constructive. We follow the [Contributor Covenant](https://contributor-covenant.org).
